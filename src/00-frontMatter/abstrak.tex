\vspace*{0.5cm}

\noindent Pengembangan perangkat lunak ini bertujuan untuk mendigitalkan proses manajemen kehadiran guru yang selama ini masih dilakukan manual oleh Kepala Bidang atau Kabid di Pondok Modern Al-Ghozali yang membutuhkan sistem efisien untuk mengolah data \textit{fingerprint} dan perizinan. Pelaksana kerja praktik berperan sebagai Project Manager sekaligus Backend Developer dengan metodologi pengembangan \textit{Scrum} menggunakan siklus \textit{sprint} mingguan. Pekerjaan utama mencakup perencanaan \textit{timeline}, pengembangan \textit{backend} menggunakan \textit{framework} Django, dan pengolahan data dengan \textit{library} Pandas. Hasil proyek adalah aplikasi web fungsional dengan fitur \textit{dashboard} statistik, pemrosesan file XLS otomatis, dan integrasi perizinan via WhatsApp yang memberikan pengalaman nyata dalam mengelola tim teknis dan mengimplementasikan logika bisnis kompleks serta relevan dengan mata kuliah Kecerdasan Artifisial, Sains Data Dasar, dan Pemrograman Berbasis Platform.

\vspace*{0.2cm}

\noindent \textbf{Kata kunci:} \textit{backend development}, Django, \textit{project management}.

\vspace*{0.5cm}

\noindent Pengembangan perangkat lunak ini bertujuan untuk mendigitalkan proses manajemen kehadiran guru di Pondok Modern Al-Ghozali. Sebelumnya, pengolahan data absensi dari mesin fingerprint dilakukan secara manual sehingga memerlukan waktu yang lama dan kurang efisien. Selain itu, proses perizinan guru masih dilakukan melalui koordinasi langsung yang berpotensi menimbulkan keterlambatan informasi.

Pelaksana kerja praktik berperan sebagai Project Manager sekaligus Fullstack Developer. Metodologi pengembangan yang digunakan adalah Scrum dengan siklus sprint mingguan. Pekerjaan utama mencakup perencanaan timeline proyek, pengembangan backend menggunakan framework Django, serta pengolahan dan analisis data menggunakan library Pandas.

Hasil dari proyek ini adalah sebuah aplikasi web fungsional yang memiliki fitur dashboard statistik kehadiran, pemrosesan file XLS secara otomatis, serta integrasi sistem notifikasi perizinan melalui WhatsApp. Proyek ini memberikan pengalaman nyata dalam mengelola tim teknis dan mengimplementasikan logika bisnis yang kompleks, serta relevan dengan mata kuliah Rekayasa Perangkat Lunak, Basis Data, dan Proyek Perangkat Lunak.

\vspace*{0.2cm}

\noindent \textbf{Kata kunci:} backend development, Django, project management, kerja praktik.
